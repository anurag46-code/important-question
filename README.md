# Important Question Website

A fun interactive website that asks "Do you love me?" with playful animations where the "No" button tries to escape when hovered over.

## Features

- ✨ Beautiful gradient background
- 💕 Floating heart animations
- 🏃‍♂️ "No" button that runs away when you try to click it
- 🎉 Celebration animation when "Yes" is clicked
- 📱 Mobile responsive design

## Deployment

Automatically deployed to Vercel on push to main branch.
